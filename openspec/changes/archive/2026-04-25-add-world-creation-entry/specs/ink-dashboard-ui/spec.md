## MODIFIED Requirements

### Requirement: 水墨工作台主布局
系统 SHALL 在应用根路径呈现一个复刻参考图结构的织世录水墨工作台界面，并提供进入创建世界页面的入口。

#### Scenario: 桌面端呈现三栏工作台
- **GIVEN** 用户使用宽屏桌面视口访问应用根路径
- **WHEN** 页面加载完成
- **THEN** 系统 MUST 显示左侧深色导航栏、中部主内容区和右侧信息栏，并保持三栏之间边界清晰

#### Scenario: 主内容区模块完整
- **GIVEN** 用户访问应用根路径
- **WHEN** 页面加载完成
- **THEN** 系统 MUST 在主内容区显示顶部世界标题横幅、统计概览、运行状态、近期事件、最新故事章节和活跃角色模块

#### Scenario: 右侧信息栏模块完整
- **GIVEN** 用户访问应用根路径
- **WHEN** 页面加载完成
- **THEN** 系统 MUST 在右侧信息栏显示当前时间、紧张关系、待解秘密和近期趋势模块

#### Scenario: 侧栏提供新建世界入口
- **GIVEN** 用户访问应用根路径
- **WHEN** 左侧深色导航栏渲染完成
- **THEN** 系统 MUST 显示可进入 `/worlds/new` 的“新建世界”入口

### Requirement: 静态世界信息展示
系统 SHALL 使用静态 mock 数据或前端临时创建状态填充工作台中的世界、角色、事件和运行信息。

#### Scenario: 显示默认世界基本信息
- **GIVEN** 用户直接访问工作台且没有创建世界临时状态
- **WHEN** 顶部横幅渲染完成
- **THEN** 系统 MUST 显示默认世界名“烟雨江南”、世界描述、分类标签和顶部操作按钮

#### Scenario: 显示创建后的世界基本信息
- **GIVEN** 用户从创建世界页面完成创建后进入工作台
- **WHEN** 顶部横幅渲染完成
- **THEN** 系统 MUST 显示用户填写的世界名称、背景简介和世界类型标签

#### Scenario: 显示统计概览
- **GIVEN** 用户访问工作台
- **WHEN** 统计概览渲染完成
- **THEN** 系统 MUST 显示角色、势力、地点、事件和秘密五类统计项，并为每类统计项显示图标、数字和单位

#### Scenario: 显示运行状态
- **GIVEN** 用户访问工作台
- **WHEN** 运行状态模块渲染完成
- **THEN** 系统 MUST 显示“世界运行中”、连续运行天数、进度条、下个时间点和暂停运行操作

#### Scenario: 显示叙事内容
- **GIVEN** 用户访问工作台
- **WHEN** 事件和故事模块渲染完成
- **THEN** 系统 MUST 显示近期事件列表、最新故事章节标题、章节摘要、章节标签、分页点和继续阅读操作

#### Scenario: 显示角色与关系信息
- **GIVEN** 用户访问工作台
- **WHEN** 角色与右侧栏模块渲染完成
- **THEN** 系统 MUST 显示活跃角色卡、紧张关系条目、待解秘密优先级和近期趋势方向
