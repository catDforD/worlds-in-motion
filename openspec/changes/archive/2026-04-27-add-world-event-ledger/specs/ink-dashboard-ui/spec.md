## MODIFIED Requirements

### Requirement: 静态世界信息展示
系统 SHALL 使用默认 mock 数据、前端临时创建状态、本地世界设定、本地世界内容种子或本地世界运行记录填充工作台中的世界、角色、事件和运行信息；角色、关系和统计项 MUST 优先读取用户保存的本地内容种子；当前时间、运行状态、近期事件和最新故事章节 MUST 优先读取本地世界运行记录。

#### Scenario: 显示默认世界基本信息
- **GIVEN** 用户直接访问工作台且没有创建世界临时状态
- **WHEN** 顶部横幅渲染完成
- **THEN** 系统 MUST 显示默认世界名“烟雨江南”、世界描述、分类标签和顶部操作按钮

#### Scenario: 显示创建后的世界基本信息
- **GIVEN** 用户从创建世界页面完成创建后进入工作台
- **WHEN** 顶部横幅渲染完成
- **THEN** 系统 MUST 显示用户填写的世界名称、背景简介和世界类型标签

#### Scenario: 显示统计概览
- **GIVEN** 用户访问工作台
- **WHEN** 统计概览渲染完成
- **THEN** 系统 MUST 显示角色、势力、地点、事件和秘密五类统计项，并为每类统计项显示图标、数字和单位

#### Scenario: 统计概览读取本地内容种子
- **GIVEN** 用户已经保存角色、势力、地点或关系种子
- **WHEN** 统计概览渲染完成
- **THEN** 系统 MUST 使用本地角色、势力和地点种子的数量更新对应统计项，并保持事件和秘密统计使用现有默认展示或可读回退

#### Scenario: 显示运行状态
- **GIVEN** 用户访问工作台
- **WHEN** 运行状态模块渲染完成
- **THEN** 系统 MUST 显示当前运行或暂停状态、连续运行天数、进度条、下个时间点和暂停或继续运行操作

#### Scenario: 运行状态读取本地运行记录
- **GIVEN** 本地世界运行记录中存在运行天数和暂停状态
- **WHEN** 运行状态模块渲染完成
- **THEN** 系统 MUST 使用本地运行天数和暂停状态更新运行状态文案与操作按钮

#### Scenario: 暂停运行按钮切换状态
- **GIVEN** 工作台已加载本地世界运行记录
- **WHEN** 用户点击暂停或继续运行操作
- **THEN** 系统 MUST 切换本地世界运行记录的暂停状态，并立即在运行状态模块中反映结果

#### Scenario: 当前时间读取本地运行记录
- **GIVEN** 本地世界运行记录中存在当前世界时间
- **WHEN** 右侧信息栏当前时间模块渲染完成
- **THEN** 系统 MUST 显示本地世界运行记录中的当前世界时间，而不是固定 mock 时间

#### Scenario: 显示叙事内容
- **GIVEN** 用户访问工作台
- **WHEN** 事件和故事模块渲染完成
- **THEN** 系统 MUST 显示近期事件列表、最新故事章节标题、章节摘要、章节标签、分页点和继续阅读操作

#### Scenario: 叙事内容读取本地运行记录
- **GIVEN** 本地世界运行记录中存在事件列表或最新章节
- **WHEN** 近期事件和最新故事章节模块渲染完成
- **THEN** 系统 MUST 优先使用本地事件列表和最新章节展示叙事内容

#### Scenario: 手动记录事件入口可用
- **GIVEN** 用户正在查看工作台近期事件模块
- **WHEN** 用户通过轻量入口填写并提交事件
- **THEN** 系统 MUST 将事件保存到本地世界运行记录，并在近期事件模块中显示新事件

#### Scenario: 推进一日入口可用
- **GIVEN** 用户正在查看工作台运行状态或当前时间模块
- **WHEN** 用户执行“推进一日”
- **THEN** 系统 MUST 更新本地世界运行记录中的当前世界时间和运行天数，并刷新工作台展示

#### Scenario: 显示角色与关系信息
- **GIVEN** 用户访问工作台
- **WHEN** 角色与右侧栏模块渲染完成
- **THEN** 系统 MUST 显示活跃角色卡、紧张关系条目、待解秘密优先级和近期趋势方向

#### Scenario: 角色模块读取本地角色种子
- **GIVEN** 用户已经保存一个或多个角色种子
- **WHEN** 活跃角色模块渲染完成
- **THEN** 系统 MUST 使用本地角色种子的姓名、身份、目标和状态展示角色卡，并为缺失头像使用现有头像占位或默认视觉

#### Scenario: 紧张关系读取本地关系种子
- **GIVEN** 用户已经保存一个或多个关系种子
- **WHEN** 紧张关系模块渲染完成
- **THEN** 系统 MUST 使用本地关系种子的 A、B、关系描述、紧张度和备注展示关系条目

#### Scenario: 无本地内容种子时保持默认展示
- **GIVEN** 用户没有保存任何内容种子
- **WHEN** 工作台角色、关系和统计模块渲染完成
- **THEN** 系统 MUST 继续使用现有默认 mock 数据或可读占位内容，而不是显示空白模块或报错

#### Scenario: 无本地运行记录时保持默认展示
- **GIVEN** 用户没有保存任何本地世界运行记录
- **WHEN** 工作台运行状态、当前时间、近期事件和最新故事章节模块渲染完成
- **THEN** 系统 MUST 继续使用现有默认 mock 数据或可读占位内容，而不是显示空白模块或报错

#### Scenario: 本地内容种子不改写运行态模块
- **GIVEN** 用户保存了角色、势力、地点或关系种子
- **WHEN** 工作台重新渲染
- **THEN** 系统 MUST NOT 根据这些种子伪造或重写运行状态、当前时间、近期事件、最新故事章节、待解秘密或近期趋势模块

#### Scenario: 本地运行记录不调用后端
- **GIVEN** 用户在工作台暂停运行、继续运行、推进一日或记录事件
- **WHEN** 系统更新工作台展示
- **THEN** 系统 MUST NOT 调用后端接口、写入数据库或触发 Agent 世界运行
